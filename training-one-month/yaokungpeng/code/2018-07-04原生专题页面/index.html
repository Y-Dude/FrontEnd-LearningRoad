为什么小学要学文言文

  1. 文言文是小升初语文的必考点，也是高考必考点的一部分
  2. 文言文是阅读古文经典的重要工具
  3. 文言文需要大量的积累，良好的文言基础，有助于语文学习
  4. 趣味文言培养孩子对语文学习的兴趣

文言文学习有难点？

趣味学习，打破文言文难点

  • 从寓言故事学文言搞定语言词语
  • 从神话故事读文言解决翻译大难题
  • 从历史故事品文言玩转文言句式
